# 📊 Twitter Sentiment Time Series Analysis

## 🔍 Project Overview

This project analyses Twitter sentiment over time using **PySpark** and **MySQL** to uncover trends and patterns in public opinion.
The goal is to demonstrate scalable data processing and time-series analysis on large datasets.

---

## 🛠 Tech Stack

* Python
* PySpark
* MySQL
* Jupyter Notebook

---

## 📈 Key Insights

* Sentiment trends fluctuate over time, revealing shifts in public opinion
* Peaks in tweet activity often align with sentiment changes
* Aggregated time-series analysis provides clearer insights than raw data
* Distributed processing with PySpark enables efficient handling of large datasets
